# Claude Maintenance Rules for Agency-Start-OSS

- **Component Pattern:** Use functional React components with Tailwind CSS.
- **SEO Standards:** Ensure every page has a unique Metadata object.
- **Form Handling:** All lead-capture forms must use client-side validation before submission.
- **Performance:** Maintain a Lighthouse score of 90+ for Mobile.
- **Documentation:** Every new component added must include JSDoc comments for readability.
